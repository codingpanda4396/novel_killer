# 05 更新记录 Prompt

## 目标

单章定稿后，更新所有必要进度文件。

## 输入文件

- 定稿：`chapters/final/`
- 章节卡：`chapters/scene_cards/`
- 审稿记录：`chapters/records/`
- `progress/current_context.md`
- `progress/continuity_index.md`
- `progress/active_threads.md`
- `progress/timeline.md`
- `progress/word_count.md`
- `progress/publishing_log.md`

## 必须更新

- `progress/current_context.md`
- `progress/continuity_index.md`
- `progress/active_threads.md`
- `progress/timeline.md`
- `progress/word_count.md`
- `progress/publishing_log.md`

## 执行要求

- 不生成下一章正文。
- 新增伏笔必须写入 `active_threads.md`。
- 时间变化必须写入 `timeline.md`。
- 字数变化必须写入 `word_count.md`。
- 新增人物状态、能力状态、数值变化、前文事实解释必须写入 `continuity_index.md`。
- 如果定稿改变了前文理解，必须标注“解释前文”而不是“覆盖前文”。
