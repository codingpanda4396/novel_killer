# 03 审稿 Prompt

## 目标

对单章草稿进行结构、连续性、商业性、人物、能力、禁区检查。

## 输入文件

- 对应草稿：`chapters/drafts/`
- 对应章节卡：`chapters/scene_cards/`
- `progress/current_context.md`
- `progress/active_threads.md`
- `progress/timeline.md`
- `bible/04_forbidden_rules.md`

## 审稿输出

```markdown
# 第X章 审稿记录

## 结论

## 必须修改

## 建议修改

## 连续性风险

## 是否可进入定稿
```

## 输出位置

- `chapters/records/第XXX章_审稿记录.md`
