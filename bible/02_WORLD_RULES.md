# 02 世界观与能力规则

## 人生余额五个基础数值

### 1. 寿命余额

显示一个人大致还能活多久。

表现形式：

```text
寿命余额：72年
寿命余额：9小时
寿命余额：00:03:17
```

规则：

- 危机临近时才显示精确倒计时。
- 平时显示模糊数值，避免信息过载。
- 不让男主随便看见所有人的死亡时间，否则剧情失控。

### 2. 财运余额

显示财富机会、资源、人脉、判断力、时机，不等于直接中奖。

表现形式：

```text
财运余额：低
财运趋势：快速上升
财运异常：非自然转移
```

规则：

- 男主可以利用信息差赚钱，但不能无限暴富。
- 每次靠能力赚钱都要伴随代价或风险。

### 3. 情缘余额

显示一个人建立真实亲密关系的能力和状态。

表现形式：

```text
情缘余额：17
情缘余额：-8
情缘状态：被掠夺
```

用于直播、网红、PUA、相亲、婚恋、豪门婚姻等单元。不写低俗性描写，重点写情感操控、孤独、错付、虚假亲密关系。

### 4. 遗憾值

显示一个人未完成执念或内心痛苦强度。

```text
遗憾值：91
最大遗憾：未解锁
```

遗憾值是单元故事发动机。完成关键遗憾后，男主可获得命运点或能力升级。

### 5. 厄运倒计时

显示即将发生的重大负面事件。

```text
厄运倒计时：00:03:17
风险类型：坠落 / 车祸 / 背叛 / 命运回收
```

规则：

- 倒计时是章节钩子神器，不能滥用。
- 重要倒计时必须在几章内兑现。
- 倒计时可以误导，但不能欺骗读者。

## 能力升级路线

### 第一阶段：人生余额可视

看见基础数值、明显倒计时，无法查看深层原因。干预重大死亡事件会消耗自身寿命。

### 第二阶段：遗憾回溯

看见对方人生中最深遗憾的片段。片段不完整，必须靠现实调查补全。连续使用会记忆混淆。

### 第三阶段：命运节点

看见未来24小时内几个关键选择节点，只能看见节点，不能看见完整未来。

### 第四阶段：余额转移感知

看见某个人的余额是否来自他人，看见转移痕迹，但不知道完整链条。

### 第五阶段：命运审计

看见一整条命运交易链，可标记异常协议，可短暂冻结非法转移，但会被天平会系统反向定位。

### 第六阶段：规则重写

终局能力。可重写天命池部分规则，作废欺诈性协议，归还被强制夺走的余额。代价是承受全城遗憾反噬，并失去大部分能力，只保留“遗憾感知”。

## 命运点规则

来源：

1. 完成他人关键遗憾。
2. 阻止非法命运转移。
3. 让当事人真正知情并重新选择。
4. 解开天平会隐藏协议。

用途：

1. 延长自身寿命余额。
2. 解锁能力。
3. 短暂冻结他人厄运。
4. 查看命运节点。
5. 对抗天平会追踪。

限制：

- 不能凭空创造命运点。
- 不能无限复活。
- 不能随便给自己加钱、加寿命。
- 每次重大改命都要有代价。

## 天平会规则

表面身份：高端财富管理公司、心理咨询中心、人生规划平台、慈善基金、创业孵化器、高端相亲机构。

真实业务：财运抵押、寿命转移、情缘掠夺、事业运置换、命运债务继承、命运协议包装。

组织口号：

> 命运本就不公平，我们只是让不公平明码标价。

天平会最危险之处不是强迫所有人签约，而是让痛苦中的人以为自己“自愿”。

## 命运协议规则

合法表象：

```text
人生规划协议
高端咨询服务协议
财富跃迁协议
情感修复协议
家庭保障协议
健康风险转移协议
```

真实内容：用寿命换钱、用财运换爱情、用情缘换流量、用未来机会换眼前成功、用父母寿命换孩子前途、用普通人厄运换权贵好运。

## 过审型表达

优先使用：命运算法、人生余额、命运协议、命运交易、命运审计、天命池、余额转移、数据异常、风险倒计时、非自然转移、欺诈性协议。

少用或不用：鬼、神棍、阴间、诅咒、作法、邪教、夺舍、灵魂买卖。

## 绝对禁止的能力失控

禁止：

- 男主想看谁死期就看谁死期。
- 男主随便买股票彩票暴富。
- 男主随便改命无代价。
- 男主复活死人。
- 女主死亡后靠外挂复活。
- 反派一挥手让全城人死亡。
- 能力设定频繁吃书。

每次能力升级必须提前埋伏笔。
