# 03 人物设定圣经

## 男主：陈默

### 基本信息

- 年龄：20岁
- 身份：软件工程专业大学生
- 家境：普通偏紧，母亲独自支撑家庭，父亲多年失踪
- 初始状态：缺钱、不甘平庸、有野心、有强烈向上欲望
- 核心能力：人生余额可视

### 性格关键词

聪明、敏感、现实、有一点功利、反应快、不轻易认命、嘴硬心软、遇事先算代价但最后常常选择救人。

### 初始欲望

> 我不想再被命运安排，我要靠自己翻身。

### 深层恐惧

- 害怕自己一辈子普通。
- 害怕母亲继续吃苦。
- 害怕父亲的失踪代表抛弃。
- 害怕自己救不了真正重要的人。
- 害怕自己最后也变成利用别人的人。

### 成长弧光

```text
只想赚钱
→ 发现钱背后有人被收割
→ 开始救人
→ 发现救人有代价
→ 发现父亲的债
→ 对女主产生复杂情绪
→ 经历失败
→ 建立团队
→ 对抗天平会
→ 获得命运规则重写权
→ 放弃支配他人命运
```

### 说话风格

前期偏现实，带一点讽刺。遇到危险时冷静，喜欢用短句。情感表达嘴硬，不直接煽情。

示例：

```text
“我不是好人，我只是刚好看见了。”
“如果命真的有价格，那穷人是不是连活着都要分期付款？”
“我怕死，但我更怕一辈子都在算值不值。”
```

### 禁止写法

禁止圣母化、无脑热血、突然变霸总、只靠外挂不动脑、遇事全靠女主解释、一直嘴炮不行动。

## 女主：林见鹿

### 基本信息

- 年龄：22—24岁
- 身份：心理学研究生 / 心理咨询实习师
- 气质：冷静、理性、白月光感
- 隐藏设定：小时候原定死亡，被陈默父亲用命运余额救下

### 性格关键词

理性、克制、清冷、温柔但有边界、对痛苦敏感、善于看见人的心理创伤。

### 初始欲望

> 弄清自己为什么总觉得“活着像欠了什么”。

### 深层恐惧

- 害怕自己的人生建立在别人牺牲上。
- 害怕自己成为陈默的负担。
- 害怕被救意味着永远欠债。
- 害怕自己没有真正选择过人生。

### 人物功能

1. 第一章被救下，形成开局钩子。
2. 作为心理学角色，帮助男主理解“遗憾值”。
3. 作为情感锚点，防止男主只把人当数据。
4. 作为主线钥匙，牵出陈远山旧案。
5. 在后期主动牺牲，推动终局选择。

### 说话风格

克制，不多话。常用反问。善于把问题拉回人的痛苦与选择。情绪爆发少，但一旦爆发要有力量。

示例：

```text
“自愿不等于清醒。被逼到没有退路，也不叫选择。”
“你看到的是余额，我看到的是人为什么会把自己逼到那一步。”
“我不想让你一直替所有人还债。”
```

### 与陈默关系节奏

1. 怀疑：她不相信陈默能看见倒计时。
2. 观察：她发现陈默确实在救人。
3. 合作：她帮助陈默分析遗憾。
4. 疏远：得知自己与陈默父亲的债有关。
5. 回归：她选择不是因为债，而是因为陈默本人。
6. 并肩：两人共同对抗天命池。
7. 自由：结尾不再由债务决定关系。

### 禁止写法

禁止花瓶、只负责被救、无脑相信男主、过早表白、靠撒娇推动剧情、用低俗暧昧制造感情线。

## 男二：周野

- 身份：陈默室友
- 方向：短视频、自媒体、直播运营
- 性格：外向、嘴碎、讲义气、现实但有底线

人物功能：

- 提供轻松感。
- 连接直播、网红、舆论、短视频单元。
- 帮男主把真相传播出去。
- 中后期被夺运，制造兄弟线危机。

台词示例：

```text
“序哥，你这不是开挂，你这是服务器权限泄露。”
“我负责让全网知道，他们负责破防。”
```

禁止纯搞笑工具人、无脑拖后腿。他可以鲁莽，但关键时刻必须讲义气。

## 女二：沈微澜

- 身份：天平会高层之女 / 命运交易体系继承者
- 气质：冷艳、危险、理性
- 立场：交易可以存在，但欺骗和强制不可接受

人物功能：

- 代表反方价值观中的合理部分。
- 诱惑男主理解天平会。
- 提供内部情报。
- 后期背叛天平会，但不完全洗白。

核心理念：

> 命运交易不是罪，真正的罪是让人以为自己没有付出代价。

禁止写成恶毒女配、倒贴女二、无意义三角恋角色。她的魅力来自价值观冲突。

## 终极反派：顾沉舟

- 身份：天平会会长
- 表面：慈善家、投资人、人生规划集团董事长
- 实质：命运交易系统操盘者

核心理念：

> 普通人不是被夺走命运，而是主动把命运卖给欲望。我只是让交易变得高效。

顾沉舟不能是无脑坏人。他必须能说服一部分读者，让读者短暂觉得“他说得好像也有道理”。

常用论点：

1. 世界本来不公平。
2. 穷人也会用命换钱，只是以前没人把价格写出来。
3. 父母本来就在为孩子消耗生命。
4. 爱情、事业、健康本来就无法兼得。
5. 天平会只是让代价透明化。
6. 大多数人痛苦不是因为被交易，而是因为没有资格上桌。

失败原因：他真正的问题不是交易，而是欺骗、垄断和强制。

禁止疯狂大笑、无意义杀人、智商掉线、最后临时忏悔洗白。

## 父亲：陈远山

- 身份：陈默父亲，第一批命运可视者
- 表面：多年失踪，被家人误以为逃避责任
- 真相：被困在天命池，成为系统核心锚点

人物功能：

- 牵出陈默能力来源。
- 解释林见鹿为何活下来。
- 解释陈默家庭命运不顺。
- 提供终局亲情高潮。

终局台词方向：

```text
“儿子，别把救我当成你的人生任务。”
“你不是我的债主，你是我的孩子。”
```

## 主角团补充角色

### 程砚

失业程序员，后加入团队。负责技术追踪、数据库入侵、系统分析。来源单元：事业运被上司偷走。

### 许棠

前天平会外围人员。负责内部规则、协议结构、交易流程。存在卧底嫌疑，可制造团队不信任。
