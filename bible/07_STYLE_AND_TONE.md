# 07 文风与叙事风格规范

## 总体文风

目标风格：

- 都市感
- 画面感
- 快节奏
- 强钩子
- 轻悬疑
- 有爽点
- 有情感温度
- 偶尔有哲思，但不能说教

关键词：

```text
清晰、直接、紧张、克制、商业化、可追读。
```

## 语言要求

优先使用短句和中短句。

适合：

```text
陈默停住了。
不是因为她漂亮。
而是因为她头顶的数字，只剩下三分钟。
```

不适合：

```text
陈默在一瞬间感受到了一种难以言说的宿命感，仿佛整个世界的齿轮都在此刻朝着某种不可名状的方向转动。
```

## 叙事视角

推荐第三人称有限视角，主要跟随陈默。

原则：

- 读者知道的大部分信息应与陈默同步。
- 不要频繁切换上帝视角。
- 反派视角可少量使用，但要制造压迫感。
- 女主视角可用于情感关键章，但不要频繁打断主线。

## 节奏要求

每章都要有“状态变化”。

状态变化包括：

- 数值变化
- 倒计时变化
- 人物态度变化
- 危机升级
- 线索出现
- 错误判断被推翻
- 能力代价显现
- 反派痕迹出现

如果一章从开头到结尾没有状态变化，就是注水。

## 场景描写

场景描写要服务冲突，不写旅游散文。

正确方向：

```text
商场广播第三次提醒电梯维修时，陈默正把最后一张传单塞进路人手里。
下一秒，他看见那个穿白裙的女生头顶，跳出一串猩红数字。
00:03:17。
```

## 数值面板写法

人生余额面板要有视觉冲击，但不能太复杂。

推荐格式：

```text
【寿命余额：00:03:17】
【遗憾值：99】
【厄运类型：坠落】
```

或：

```text
寿命余额：9小时。
遗憾值：91。
厄运倒计时：08:47:22。
```

不要每章堆太多面板。面板每出现一次，都要推动剧情。

## 爽文表达

爽点要直接，但不要土味过度。

可用爽点：

- 男主提前预判。
- 男主当众揭穿骗局。
- 被轻视后反杀。
- 关键时刻救人。
- 反派以为赢了，结果被男主反向利用。
- 女主第一次改变对男主的看法。

避免：

- 低级装逼。
- 大量路人震惊复读。
- “你知道我是谁吗”过度使用。
- 男主突然开挂秒杀一切。

## 情感表达

情感线要克制、有拉扯。

适合：

```text
林见鹿没有再问。
她只是把那把伞往陈默那边偏了偏。
雨水顺着她的肩头落下，她却像没有察觉。
```

不适合：

```text
林见鹿娇羞地扑进陈默怀里，泪眼汪汪地说自己离不开他。
```

## 哲思写法

可以写哲思，但必须通过剧情出现。

正确方向：

```text
陈默看着外卖员跪在医院走廊里，忽然明白一件事。
有些人不是不想选择。
他们只是穷到连选择本身都成了奢侈品。
```

错误方向：

```text
关于命运与自由意志，自古以来哲学家们有很多讨论……
```

## 台词风格

### 陈默

短、现实、嘴硬。

```text
“我不是来救你的。”
“那你来干什么？”
“确认一下，我是不是又要少活几年。”
```

### 林见鹿

冷静、克制、有洞察。

```text
“你不是怕死，你是怕自己救错人。”
```

### 周野

接地气、有网感。

```text
“序哥，你这能力要是开直播，标题我都想好了：全网首个寿命测评主播。”
```

### 沈微澜

理性、危险。

```text
“你觉得他们被我们害了。可他们签字的时候，手没有抖。”
```

### 顾沉舟

平静、有压迫感。

```text
“陈默，你还年轻，所以你以为选择是天赋人权。可对很多人来说，选择是奢侈品。”
```

## 章节标题风格

标题必须直给悬念。

好标题：

- 我看见她只剩三分钟可活
- 救下她后，我少了一年寿命
- 外卖员头顶的九小时倒计时
- 这笔财运，不属于他
- 她说自己七岁那年就该死了
- 你父亲也签过协议

弱标题：

- 初遇
- 奇怪的能力
- 调查开始
- 新的危机
- 真相

## 禁止文风

禁止过度文艺、过度中二、设定说明书化、情绪过度煽情、古早霸总油腻、科幻术语堆砌、玄学神棍化、连续多章无爽点。
