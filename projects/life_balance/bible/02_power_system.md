# 02 能力体系

## 文件用途

本文件用于约束人生余额、命运点、遗憾回溯、命运协议等核心能力规则，避免长期连载中能力失控。

## 能力原则

- 能力必须有边界。
- 干预命运必须有代价、风险或后续反噬。
- 前 8 万字不能解锁终局级能力。
- 数值变化必须写入进度文件，不能只在正文里出现。

## 当前能力模块

| 模块 | 用途 | 前8万字限制 |
| --- | --- | --- |
| 寿命余额 | 生死危机提示 | 可预警，不可无限续命 |
| 财运余额 | 财务风险与交易骗局提示 | 只服务案件和主线线索 |
| 情缘余额 | 关系与情感交易提示 | 谨慎使用，避免俗套恋爱外挂 |
| 遗憾值 | 单元救赎核心 | 可降低，不应每次都清零 |
| 厄运倒计时 | 章内强钩子 | 不能每章机械重复 |
| 命运点 | 阶段性奖励和升级资源 | 前 8 万字不得触达终局阈值 |

## 继承资产

- 旧版能力规则：`bible/02_WORLD_RULES.md`
- 连续性记录模板：`bible/09_CONTINUITY_TRACKER.md`

## 系统字段

- 文件职责：能力和代价的规则入口。
- 更新频率：中，新增能力阶段前必须更新。
- 使用阶段：章节卡生成、正文生成、连续性检查。
- 禁止写入：临时灵感、未确认能力、正文片段。
