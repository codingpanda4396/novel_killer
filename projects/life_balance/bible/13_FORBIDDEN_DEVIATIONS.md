# 13 禁止偏离清单

这个文件用于防止 AI 写作失控。

## 世界观禁止偏离

禁止：

1. 把人生余额写成修仙灵力。
2. 把天平会写成邪教组织。
3. 加入鬼神、妖魔、地府、轮回审判等新体系。
4. 加入外星人、平行宇宙、时间穿越等新大设定。
5. 让系统突然拟人化卖萌。
6. 让男主进入无限副本。
7. 把都市文写成玄幻打怪升级。
8. 让命运余额可以随便买卖且无代价。
9. 让能力变成万能许愿机。
10. 让结局变成统治世界。

## 人物禁止偏离

### 陈默

禁止：

- 变成纯圣母。
- 变成冷血利己主义到底。
- 变成无脑霸总。
- 变成只会嘴炮不行动。
- 变成随便开后宫的男主。
- 忘记他最初的现实欲望：赚钱、翻身、自由。

### 林见鹿

禁止：

- 变成花瓶。
- 变成恋爱脑。
- 变成只会被绑架、被救。
- 变成无条件相信男主。
- 变成工具型解说员。
- 过早告白。

### 周野

禁止：

- 只负责搞笑。
- 强行泄密拖后腿。
- 写成无脑贪财小人。

### 沈微澜

禁止：

- 变成恶毒女配。
- 变成倒贴女二。
- 变成无意义三角恋角色。

### 顾沉舟

禁止：

- 变成疯批小丑。
- 变成无脑反派。
- 最后突然忏悔洗白。
- 因为剧情需要突然智商下线。

## 剧情禁止偏离

禁止：

1. 前50章解释完整世界观。
2. 前100章揭露父亲完整真相。
3. 第二卷前出现天命池。
4. 第三卷前让顾沉舟全面下场。
5. 第四卷前启动全城命运审计。
6. 第五卷前公开父亲还活着。
7. 第六卷再开新地图。
8. 中途临时增加更大幕后黑手。
9. 结局留下主线不收。
10. 完结时让男主无代价成神。

## 能力禁止偏离

禁止：

- 男主无代价救所有人。
- 男主随便给自己加寿命。
- 男主靠彩票股票无限暴富。
- 男主能复活死人。
- 男主能完全预测未来。
- 男主能读取所有人内心。
- 男主能力每章变一个新功能。
- 能力升级没有铺垫。
- 命运点凭空增加。
- 倒计时出现后长期不兑现。

## 情感线禁止偏离

禁止：

- 10章内确认恋爱关系。
- 频繁撒糖干扰主线。
- 用误会拖100章。
- 用第三者狗血推进。
- 女主用牺牲证明爱男主过多次。
- 男主把女主当债务对象直到结尾。

情感线核心：

> 从命运债务到自由选择。

## 商业节奏禁止偏离

禁止：

- 连续三章无爽点。
- 连续三章无主线推进。
- 连续两章纯解释。
- 连续两章纯心理活动。
- 连续五章主角没有主动行动。
- 每章开头都慢热铺垫。
- 每章结尾都没有钩子。
- 单章引入太多新人物。
- 单章解释太多新设定。

## 过审禁止偏离

禁止：

- 露骨色情。
- 血腥细节。
- 邪教仪式。
- 真实机构违法影射。
- 现实政治阴谋。
- 自杀方法细节。
- 仇恨群体表达。
- 未成年人不当情节。

## 完结禁止偏离

第六卷必须完成：

- 顾沉舟败局。
- 天命池规则重写。
- 欺诈协议作废。
- 陈远山线收束。
- 林见鹿脱离命运债务。
- 陈默失去大部分能力。
- 男女主自由选择彼此。
- 人生不能被定价主题落地。

禁止：

- 顾沉舟逃走开启第二部。
- 天命池背后还有更大系统。
- 陈默成为新的天平会会长。
- 林见鹿死亡后复活。
- 男主全能力保留无代价。
- 大结局突然引入新反派。
