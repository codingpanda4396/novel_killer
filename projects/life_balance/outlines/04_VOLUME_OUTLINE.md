# 04 六卷详细大纲

## 第一卷：我看见她只剩三分钟可活

卷目标：让读者迅速明白男主能看见人生余额、开局救下女主、救人有代价、城市里有人在交易人生余额。

主要剧情：陈默在商场兼职时，看见林见鹿头顶只剩三分钟死亡倒计时。他强行把她从电梯前拉开，电梯坠落。随后在外卖员、小女孩、彩票站、林见鹿连续厄运中发现，人生余额不是单纯外挂，而是一套有代价的系统。

卷内单元：

1. 电梯倒计时：开局救女主。
2. 外卖员最后一单：遗憾值与救赎。
3. 彩票店财运异常：赚钱爽点转向阴谋。
4. 林见鹿死亡回环：女主主线启动。

卷末爆点：

```text
寿命余额：30天。
遗憾值：100。
命运权限：未觉醒。
```

---

## 第二卷：命运交易所

卷目标：把单个异能故事升级成城市阴谋。

主要剧情：陈默为了补回寿命，与林见鹿合作调查天平会。他们连续碰到女主播情缘掠夺、校园贷财运抵押、高端相亲局等案件，最终追查到“白塔人生咨询中心”。在白塔中心档案室，他们发现林见鹿原本七岁就该死亡，当前寿命来源指向陈远山。

卷内单元：

1. 女主播情缘余额：流量与虚假亲密。
2. 校园贷财运抵押：大学生创业与成功学骗局。
3. 高端相亲局：情缘余额筛选。
4. 白塔中心：命运档案室。

卷末爆点：

```text
林见鹿，原定死亡年龄：7岁。
当前寿命来源：未知转移。
债权人：陈远山。
```

---

## 第三卷：我父亲卖掉了我的命

卷目标：将父亲旧案、女主债务、男主自身命格绑定，制造中期强冲突。

主要剧情：陈默调查父亲，发现父亲曾是第一批命运可视者，为了救林见鹿签下命运抵押协议。代价不仅是父亲自己的寿命，也包括陈默母亲的财运和陈默未来的一部分命格。沈微澜登场，带他看见天平会的“另一面”：很多人是自愿交易。陈默动摇，并在强闯天平会据点时第一次惨败。

卷内单元：

1. 失业程序员事业运被偷：职场打脸。
2. 母亲寿命协议：自愿牺牲的复杂性。
3. 沈微澜登场：反方价值观诱惑。
4. 顾沉舟初登场：男主惨败。

卷末爆点：

```text
这座城市至少三分之一的人，都被天平会标价了。
```

---

## 第四卷：全城命运审计

卷目标：男主从个人英雄升级为团队破局者，中期大高潮。

主要剧情：陈默组建团队：林见鹿、周野、程砚、许棠。团队调查医院寿命黑市、豪门婚礼情缘掠夺、城市幸运儿计划，最终让陈默觉醒“命运审计”能力。他启动全城命运审计，使大量人看见自己的命运协议真相。天平会第一次暴露，但也引发混乱。顾沉舟反击，冻结陈默人生余额。

卷内单元：

1. 医院寿命黑市：权贵续命。
2. 豪门婚礼情缘掠夺：女主感情确认。
3. 城市幸运儿计划：公益包装下的筛选。
4. 全城命运审计：中期最大场面。

卷末爆点：

```text
陈默名下所有人生余额，冻结。
下一次使用能力，他将直接进入清算程序。
```

---

## 第五卷：命运清算日

卷目标：进入终局前夜，所有人物线开始收束。

主要剧情：天平会准备启动“天命池”，将全城人生余额汇总分级。周野被夺运，林见鹿主动签下归还协议，陈默发现父亲陈远山没有死，而是被困在天命池底层，成为系统核心锚点。沈微澜背叛天平会，帮助陈默进入天命池。

卷内主线事件：

1. 周野被夺运：兄弟线危机。
2. 林见鹿归还协议：爱情线大虐。
3. 父亲还活着：亲情线反转。
4. 命运清算日：全城危机。

卷末爆点：

```text
父亲说：
“不要救我。我已经是这个系统的一部分。
你要救的，是还没被它吃掉的人。”
```

## 第六卷：人生不能被定价

卷目标：终局决战，完成价值观收束与情感落地。

主要剧情：顾沉舟公开包装“命运优化计划”，并获得大量支持。陈默进入天命池核心，面对三个选择：

1. 保留系统，林见鹿活。
2. 关闭系统，林见鹿可能回到原定死亡。
3. 重写规则，由自己承受全城遗憾反噬。

陈默选择第三条路。最终作废欺诈性协议，归还被强制夺走的人生余额，禁止命运债务继承，使命运系统不再被组织私有化。

终局规则：

```text
所有欺诈性命运协议作废。
所有强制转移的余额归还。
所有协议必须公开真实代价。
未成年人、被胁迫者、无意识者不可签约。
任何人不能继承他人命运债务。
```

结尾：陈默失去大部分能力，只保留微弱的遗憾感知。林见鹿的人生余额变成“未知”，象征她终于不再被系统计算。两人回到第一次相遇的商场。

最终台词方向：

```text
“因为这一次，不是命运安排我来救你。
是我自己想走向你。”
```
